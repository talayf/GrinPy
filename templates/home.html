GrinPy

Prepare and Analyze

About

GrinPy is a web-based graphical user interface (GUI) to prepare input files for MD simulations using GROMACS and facilitate different approaches for trajectory analysis. The purpose of this project is to provide users, especially those with little experience in command lines, with a more modern and unified platform to smoothly fulfill their tasks.

SERVICES

Preparation

Generate all input files necessary to run a molecular dynamics simulation.

Analysis

Analyze resulting trajectories using commonly used and novel approaches.
